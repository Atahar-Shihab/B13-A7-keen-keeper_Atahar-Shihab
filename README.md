## 🎯 About The Project

Life gets busy, and it's easy to lose touch with the people who matter most. **KinKeeper** is a lightweight, intuitive personal CRM (Customer Relationship Management) designed specifically for friendships. It helps you track when you last spoke to your friends, set relationship goals, and easily log your calls, texts, and video chats.

## ✨ Features

- **📊 Dashboard Overview:** View a summary of your connections, including who is "overdue", "on-track", or "almost due" for a catch-up.
- **👥 Friend Profiles:** Detailed cards for each friend displaying contact streaks, shared tags (e.g., *college*, *gym*), and customizable interaction goals.
- **⚡ Quick Check-Ins:** One-click logging for 📞 **Calls**, 💬 **Texts**, and 📹 **Video Chats** using custom UI icons.
- **🕒 Interactive Timeline:** A historical feed of all your interactions, complete with a dropdown filter to easily sort by interaction type.
- **📈 Friendship Analytics:** Beautiful interactive Pie Charts visualizing your communication habits (built with Recharts).
- **📱 Fully Responsive:** Carefully crafted with Tailwind CSS to look pixel-perfect on mobile, tablet, and desktop screens.

## 💻 Tech Stack

This project is built with modern web technologies:

* **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Routing:** [React Router DOM](https://reactrouter.com/)
* **State Management:** React Context API
* **Charts:** [Recharts](https://recharts.org/)
* **Notifications:** [React Hot Toast](https://react-hot-toast.com/)
* **Icons:** [Lucide React](https://lucide.dev/) + Custom UI Assets
